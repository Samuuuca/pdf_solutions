# Soluções em PDF de Fácil Acesso

Instale as os executaveis e tenha acesso a soluções em pdf sem precisar de sites ou aplicativos externos.

 
### Soluções

- Juntar 2 ou mais PDFs.
- Separar um PDF em Vários.
- Transformar PDF em DOCX (word)
    
    
### Instalação
	
1 - Instalação Direta: Baixe os modulos executaveis desejados pelo link

#### [Baixar Executáveis](https://github.com/Samuuuca/pdf_solutions/releases/tag/Funcional)

```shell
	# Abrir o Executar
	Use o comando WIN + R 
	
	# Caminho para a pasta
	digite shell:sendto
	
	# Adicione os Arquivos
	Adicione o Atalho para os executáveis ou adicione os executáveis diretamente
```
